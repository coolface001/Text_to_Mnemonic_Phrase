# 助记词生成器 (Minimalist Secure Mnemonic Generator)

輸入任意的詞彙 詩句 或者是任何記憶中的東西 可長可短 不限制輸入內容  它將幫你轉換成英文的24組助記詞 
便於創造錢包以及記憶資產

---

## ✨ 核心特性

* 🔒 **100% 離線本地運作**：
  * 所有密碼學計算（SHA-256 與 BIP-39 映射）完全使用瀏覽器原生的 Web Crypto API (`crypto.subtle.digest`) 於本地執行。
  * 零外部依賴、不載入任何 CDN 腳本，完全杜絕資料外洩的風險，支持完全斷網離線使用。
* 🗑️ **安全最大化銷毀機制（閱後即焚）**：
  * **生成即銷毀輸入**：點擊「生成助記詞」按鈕後，原始輸入框內容會在產出助記詞的瞬間**立即清空**，防止敏感輸入在螢幕上長時間留存。
  * **複製即銷毀顯示**：點擊「複製助記詞」後，助記詞會寫入系統剪貼簿，同時顯示框內容會**瞬間恢復為空**，將螢幕暴露時間縮至最短，防止窺視與截圖。

---

## 🛠️ 技術原理

1. **哈希計算**：利用瀏覽器原生 API 計算輸入文本的 `SHA-256` 值，產生 256 位元（bits）的熵（Entropy）。
2. **校驗和產生**：對該熵進行再次哈希計算，提取前 8 位元作為校驗和（Checksum）。
3. **單詞映射**：將 256 位熵 + 8 位校驗和組合成 264 位數據流，每 11 位分割為一組，映射至內置的 2048 個官方 BIP-39 英文單詞庫，產出 24 個助記詞。

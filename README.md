# Text to Mnemonic Phrase  
## 任意字符 → 助記詞 → 錢包生成

### 🧠 什麼是助記詞（Mnemonic Phrase）？
助記詞是一組能夠幫你記住私鑰的單字，符合 BIP39 標準，可以用來生成加密貨幣錢包（如 Bitcoin、Ethereum、Solana 等）。

---

### 🛠️ 功能簡介
- 輸入任意字串（如：一段話、詩、密碼）
- 經由 SHA256 雜湊 + BIP39 字典映射
- 得到合法的助記詞（Mnemonic Phrase）
- 可匯入任何支援 BIP39 的錢包軟體使用（如 MetaMask、Trust Wallet 等）
